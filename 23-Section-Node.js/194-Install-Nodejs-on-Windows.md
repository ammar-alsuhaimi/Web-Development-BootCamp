# Installing Node.js on Windows

## Installation Steps

1. **Download the Windows installer** from the [Node.js® website](https://nodejs.org/).
    - Choose the Long Term Support (LTS) version that’s shown on the left.

2. **Run the installer** (the `.msi` file you downloaded in the previous step).

3. **Follow the prompts in the installer**:
    - Accept the license agreement.
    - Click the NEXT button a bunch of times.
    - Accept the default installation settings.

4. **Restart your computer**:
    - You won’t be able to run Node.js® until you restart your computer.

5. **Confirm that Node has been installed successfully**:
    - Open a Hyper terminal.
    - Enter the command:
    ```sh
    node --version
    ```
    - You should see the version of Node.js you just installed.
